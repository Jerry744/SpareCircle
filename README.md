
  # SpareCircle

  SpareCircle is an open source LVGL translator, designed as a free alternative to proprietary tools such as Squareline Studio.

  ## Features

  - Open source and free to use
  - LVGL-oriented visual workflow and translator tooling
  - Built with Vite + React for fast local development

  ## Getting Started

  Install dependencies:

  ```bash
  npm install
  ```

  Start the development server:

  ```bash
  npm run dev
  ```

  Build for production:

  ```bash
  npm run build
  ```
